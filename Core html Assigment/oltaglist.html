1. DOCTYPE: The DOCTYPE is not technically a tag but a declaration that tells the browser what kind of file will be loaded. This tag tells the browser the type of HTML that will be loaded.
  2. html: The html tag comes after the DOCTYPE tag. This tag defines the document as a web page; every other element will be nested inside. The HTML tag specifies the beginning and end of an HTML document.
  3. head: The head section of an HTML document is represented by a head tag. This tag is enclosed within the html tag and gives general information about the webpage.
  4. title: The title tag declares the title of the web page. This tag is enclosed within the head tag. The title tag will appear on the title bar or the tab in the browser window but not on the actual web page.
  5. body: The tag displays all the visible content on a web page. Images, links, plain text, videos, and more can be found inside the and tags.
